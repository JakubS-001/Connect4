Projekt Connect4 w C++ (obiektowy)

Szkielet projektu przeniesiony z oryginalnej wersji C. Zawiera klasy:
- `Board` - logika planszy i sprawdzanie zwycięstwa
- `Game` - stan gry, tury, wyniki
- `SaveManager` - zapisywanie/odczyt wyników
- `UI` - interfejs oparty na Allegro (pętla gry, rysowanie)

Aby zbudować: `make`
Uruchom: `./connect4_cpp`
